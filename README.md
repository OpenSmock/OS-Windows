# OS-Windows

Extensions for Pharo-OS-Windows in OpenSmock.

## Prerequisites

Load Pharo-OS-Windows

```smalltalk
Metacello new 
	repository: 'github://astares/Pharo-OS-Windows/src';
	baseline: 'OSWindows' ;
	load.
```
